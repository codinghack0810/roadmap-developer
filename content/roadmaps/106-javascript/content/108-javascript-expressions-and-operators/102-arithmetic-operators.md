# Arithmetic operators

The Arithmetic operators perform addition, subtraction, multiplication, division, exponentiation, and remainder perations.

Arithmetic operators in JavaScript are as follows:
- `+` (Addition)
- `-` (Subtraction)
- `*` (Multiplication)
- `/` (Division)
- `%` (Remainder)
- `**` (Exponentiation)
